FOR:
Es un bucle al cual le definimos 3 parámetros:
    - primero una variable, la cual se usará para repetir el bucle.
    - después la condición que mientras devuelva el valor 'true', el bucle sigue repitiéndose.
    - por último incrementamos la variable inicialmente asignada a un valor.
ej:
    for (i = 0, i < 5, i++) {
        console.log(i);
    }
    // nos va a imprimir "0, 1, 2, 3, 4".

&& , ||, !:
Son operadores lógicos que nos sirven para comparar dos o más expresiones.
    - &&: significa "Y" (and), devuelve 'true' si todas las expresiones son 'true'.
    - ||: significa "O" (or), devuelve 'true' si alguna expresión es 'true'.
    - !: significa "NO" (not), devuelve el resultado contrario recibido.
